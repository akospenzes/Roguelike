# Roguelike

Roguelike game made by me for the BME Game Development Laboratory course(VIIIMA24). Made in Unity 2022.03.11f1

## How to play
- Movement: W, A, S, D keys
- Aim: Mouse movement
- Shoot: Space key
- Switch weapons: 1, 2, 3 keys
- Pause: Escape key
